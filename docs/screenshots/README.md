# Screenshots

Place the following screenshots in this folder:

| Filename | Description |
|----------|-------------|
| `homepage.png` | Homepage with hero section and categories |
| `search-results.png` | Search results page showing products |
| `docker-stats.png` | Terminal showing `docker stats` output |
| `endpoints.png` | Terminal showing `make endpoints` output |

## How to capture screenshots

```bash
# macOS
# Press Cmd + Shift + 4 to capture a selection

# Or use the Screenshot app
# Press Cmd + Shift + 5
```
