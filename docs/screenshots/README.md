## How to Capture Screenshots

### macOS

```bash
# Capture selection
Cmd + Shift + 4

# Capture window
Cmd + Shift + 4, then Space

# Screenshot app (more options)
Cmd + Shift + 5
```

### GIF Recording

For recording GIFs like `checkout-flow.gif`, use:
- **macOS**: [Kap](https://getkap.co/) or [LICEcap](https://www.cockos.com/licecap/)
- **Windows**: [ScreenToGif](https://www.screentogif.com/)
- **Linux**: [Peek](https://github.com/phw/peek)

## Naming Convention

- Use lowercase with hyphens: `feature-name.jpeg`
- Prefer `.jpeg` for photos/screenshots
- Use `.gif` for animations
- Use `.png` for diagrams/icons
